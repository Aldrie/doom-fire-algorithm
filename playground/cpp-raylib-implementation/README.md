# Doom-fire-effect_CPP-Raylib
![doom fire image](doom_fire.png)

# Requirements
 - g++
 - raylib 
 - Cmake 

# Compiling and running (linux)

```bash
    git clone https://github.com/FrankSteps/Doom-fire-effect_CPP-Raylib
    cd Doom-fire-effect_CPP-Raylib
    make
    make run 
```
